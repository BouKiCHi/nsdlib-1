MML reference: Relative Volume

定義（Define）

        (
        )

引数（Inputs）

無し

内容（Contents）

(
音量を1下げます。
)
音量を1上げます。

注意（Notes）

2A03の三角波、⊿PCMトラックでは音量指定はできません。

例（Sample）

    [4 c& ) ]     /* cresc. */
    [4 c& ( ]     /* decresc. */
