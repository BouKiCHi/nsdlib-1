NSDL（NES Sound Driver Library）

MML reference: Release

定義（Define）

        R@voice
        Rvvolume
        Rmmode

引数（Inputs）

voice　（`R@'命令の引数）

Key Off時の音色です。

volume　（`Rv'命令の引数）

Key Off時の音量です。

mode　（`Rm'命令の引数）

ゲートタイムによりKey Offされている期間の音の処理方法です。
休符の区間は、休符に臨時記号をつける事で以下のモードを任意に設定可能です。
つまり、ゲートタイム区間中と休符の区間では異なったモードを指定する事が可能で、 本コマンドでは、ゲートタイム区間中のモードを指定します。

0
強制的に、音量を０にします。
1
直前のノートを、R@, Rvコマンドの値で鳴らします。
2
直前のノートのエンベロープ（音色・音量）を継続します。

内容（Contents）

音の余韻の処理を設定できます。
詳細は、引数の項目を参照してください。

注意（Notes）

2A03の三角波では無効です。

例（Sample）

    Rv2 R@2 Rm1 l4 o4 v13 q8 c,8. ccc cccc     /* stacc. */
