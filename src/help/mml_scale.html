MML reference: Scale

定義（Define）

        Scale(mode)

引数（Inputs）

mode

スケールを指定します。
基本は、長調の音楽の時は"Ionian"(0)スケールを使います。
短調の時は、自然短音階である"Aeolian"(5)スケールを使う他、 和声的短音階の"Harmonic minor"(10)や、旋律的短音階の"Melodic minor"(20)を使います。
その他のスケールは、是非、研究してみてください。

10番台のスケールは和声的短音階から発展したスケールです。
20番台のスケールは旋律的短音階から発展したスケールです。

1 c d e-f g a b- Dorian
2 c d-e-f g a-b- Phrygian

4 c d e f g a b- Mixo-Lydian
5 c d e-f g a-b- Aeolian （自然短音階）
6 c d-e-f g-a-b- Locrian
7 c d+e+f+g a+b

11 c d-e-f g-a b- Locrian natural13

13 c d e-f g a b- Dorian ＃11
14 c d-e f g a-b- Harmonic minor perfect 5th below
15 c d+e f+g a b Lydian ＃9
16 c d-e-f-g-a-a Altered ♭7
20 c d e-f g a b Melodic minor （旋律的短音階）
21 c d-e-f g a b- Dorian ♭9
22 c d e f+g+a b Lydian ＃5
23 c d e f+g a b- Mixo-Lydian ＃11 (Lydian ♭7)

内容（Contents）

スケールを設定します。

注意（Notes）

ＭＭＬコンパイラーの疑似命令です。
次にK{}コマンド若しくは、Scaleコマンドが来るまで、 本コマンドで指定された音階に、自動で臨時記号が付加されてコンパイルされます。

各記述ブロック（Sub, BGM, SE）では、デフォルトは調号無し（ハ長調、アイオニアンスケール）となります。
従い、サブルーチンコールする場合は、サブルーチン先も調号・スケールを指定して下さい。

音楽理論については、別途、和声についての参考書等をご参照ください。

例（Sample）

    K{g-moll}   scale(20) gab>cdefg scale(5) agfedc/* g-moll melodic minor scale */
