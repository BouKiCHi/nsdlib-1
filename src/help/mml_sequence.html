NSDL（NES Sound Driver Library）

トップ＼ 使い方＼ MMLコンパイラ＼ MML文法＼

MML reference: Music Sequence block

定義（Define）

Sub(n){
	subroutine sequence
}

S(n){
	subroutine sequence
}

BGM(n){
	bgm sequence
}

SE(n){
	se sequence
}

引数（Inputs）

n

各ブロックの番号を指定します。

sequence

曲シーケンスのデータです。

内容（Contents）

BGM,SE,サブルーチンの演奏パターンを定義します。
シーケンスデータのサイズを最適化するためには、 ぜひ、BGM,SEデータフォーマット（Format of BGM and SE）を御一読ください。
