NSDL（NES Sound Driver Library）

使い方（Usage）

サウンドドライバー（nsd.lib）

サウンドドライバー本体です。
開発環境cc65用のオブジェクトライブラリとして開発しています。

This is sound driver.
It was developed as object library for cc65.

ＭＭＬコンパイラ（nsc.exe）

MMLコンパイラーです。
MMLで記述されたBGM, SEを、本サウンドドライバーで再生可能なデータに変換します。
アセンブリ言語・Ｃ言語のデータ配列への出力のほか（未実装）、 NSFフォーマット（プレイヤー用のデータフォーマット）への出力に対応しています。

This is MML compiler.
It can convert the MML(Music Macro Langage) to the deta for this sound driver.
An output formats are as followss.
* An array variable for the C-language and Assembly language (Now developping) .
* NSF format that is music format for some nsf player.
